## 功能简介

Mobile Coder 可以让你在手机上远程控制电脑上的 Claude Code：

- 📱 **手机控制**：在手机浏览器上操作 Claude Code
- 🖥️ **终端同步**：手机和电脑终端内容实时同步
- ⌨️ **快捷键**：支持常用快捷键输入
- 👀 **多人观看**：支持多个设备同时观看

## 准备工作

在开始之前，你需要准备：

### 1. 一台电脑（运行 Agent）

- macOS 系统
- 已安装 [Claude Code](https://github.com/anthropics/claude-code)
- 已安装 **tmux**（运行下面命令安装）：
  ```bash
  brew install tmux
  ```

### 2. 手机一部

- 安装任意浏览器（Chrome、Safari 等）
- 能访问网络（4G/5G/WiFi 均可）

## Agent 安装配置

### 步骤 1：获取 Agent 程序

从以下地址下载编译好的程序：

📦 **下载地址**：https://github.com/vinceyang/MobileCoder/releases

下载 `client-darwin-arm64`（Mac Apple Silicon）或 `client-darwin-amd64`（Mac Intel）。

下载完成后，解压得到 `client` 文件。

### 步骤 2：运行 Agent

```bash
# 赋予执行权限
chmod +x client

# 运行，连接服务端
./client -server 121.41.69.142:8080
```

启动成功后会显示绑定码：

```
==========================================
  请在 H5 页面输入以下绑定码:
==========================================
  绑定码: abc123
==========================================
```

> 💡 **记住这个绑定码**，手机连接时需要用到。

### 步骤 3：保持运行

Agent 启动后，会自动在 tmux 中运行 Claude Code。

**重要**：保持终端窗口不要关闭，关闭后 Agent 会停止运行。

## 手机端使用

### 步骤 1：打开 H5 页面

用手机浏览器打开：

```
http://121.41.69.142:3001
```

### 步骤 2：输入绑定码

1. 在绑定页面，输入 Agent 端显示的 6 位绑定码
2. 点击「绑定」按钮
3. 绑定成功后会自动进入终端页面

### 步骤 3：开始使用

现在你可以在手机上：

- 📝 **发送文本**：在输入框输入内容，点击发送
- ⌨️ **快捷键**：点击「快捷」按钮切换到快捷键模式
- ⬆️ ⬇️ **上下滑动**：浏览终端历史输出

## 常见问题

### Q: 绑定失败？

A:
1. 检查 Agent 是否正在运行
2. 检查绑定码是否正确（6 位）
3. 检查网络是否正常

---

### Q: 手机上看到的内容和电脑不一致？

A: 刷新页面即可同步最新内容。

---

### Q: 滑动不流畅？

A: 尝试在页面顶部向下拉刷新。

---

### Q: 如何关闭？

- **手机端**：直接关闭浏览器即可
- **Agent 端**：在终端按 `Ctrl+C`
